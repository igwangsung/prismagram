# prismagram
instagram clone with Express + Prisma + React + React Native

## User Stories 
- [ ] see the feed 
- [ ] like / unlike a photo
- [ ] comment on a photo
- [ ] search by user
- [ ] search by location 
- [ ] see user profile
- [ ] follow / unfollow user
- [ ] see the full photo
- [ ] edit my profile
- [ ] create account
- [ ] log in 
- [ ] upload a photo
- [ ] edit photo(Delete)
